# GiftTaxChecker-vero.fi

A program that calculates relatives gift tax from vero.fi tax brackets.

https://www.vero.fi/en/individuals/property/gifts/](https://www.vero.fi/en/individuals/property/gifts/gift-tax-calculator/






## How2Use

1. Go to the GitHub page and click the green Code button.

2. Choose Download ZIP and unzip it.

3. Open the unzipped folder. You will see GiftTaxChecker.bat.

4. Install Java if you do not have it. You can download it here:
https://adoptium.net/download?link=https%3A%2F%2Fgithub.com%2Fadoptium%2Ftemurin11-binaries%2Freleases%2Fdownload%2Fjdk-11.0.29%252B7%2FOpenJDK11U-jdk_x64_windows_hotspot_11.0.29_7.msi&vendor=Adoptium

5. After installing Java, double-click GiftTaxChecker.bat to run the program.

6. When the window opens, enter the password: First

7. Choose A for close relatives or B for distant relatives.

8. Enter the gift value when asked. The program will show the tax amount or tell you that no tax applies.

After that, close the program by pressing ANY key!
